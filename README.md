# AI Dev Tools 2025

Repository for the DataTalksClub course. This project explores the use of AI tools in software development, containing weekly projects and exercises.

## Course Modules

| Week | Project | Description | Key Technologies |
|------|---------|-------------|------------------|
| **Week 1** | [Todo App](./week_1_todo_app/) | A Django-based Todo application featuring Create, Read, Update, and Delete (CRUD) operations. Built with the assistance of AI tools to demonstrate rapid prototyping and boilerplate generation. | Python, Django, SQLite, Pico.css |
| **Week 2** | [Snake Game](./week_2_snake_game/) | A multiplayer Snake Game featuring a React frontend and FastAPI backend. Includes authentication, leaderboards, and containerized deployment. | React, FastAPI, Docker, PostgreSQL |
